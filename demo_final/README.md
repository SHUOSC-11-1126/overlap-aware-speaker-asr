# Portable Team Research Demo

Open `index.html` directly, or run a tiny local static server:

```bash
python -m http.server 8000
```

Then open `http://127.0.0.1:8000/demo_final/index.html` if you started the server from the repository root, or `http://127.0.0.1:8000/index.html` if you started it inside `demo_final/`.

This is a replay demo: all outputs are precomputed from committed research artifacts. It does not run Whisper, PyTorch, separation, LLMs, or model downloads.
