# Project Skills

These are not model-training skills. They are project continuation and engineering skill cards for future research and maintenance work.

Each skill card answers the same six questions:

- What question does this skill explore?
- Why is it relevant to the current project?
- Inputs
- Outputs
- What not to do
- Success criteria
- Owner suggestion

The point is to keep the project moving without drifting into unfocused modules. Each skill must serve one clear research question.
